# Mindful Coach

A calm, minimalist health coaching application designed for mindful wellness, goal tracking, and AI-powered health insights.

## Features

- **Onboarding Experience**: Personalize your journey by selecting primary wellness goals (Sleep, Stress, Energy, Weight).
- **Mindful Design**: A clean, balanced interface that reduces cognitive load, utilizing the "Serene Vitality" design system.
- **Energy Score**: A daily visual indicator of your wellness state.
- **Interactive Coaching**: Real-time AI chat for health guidance and session recommendations.
- **Data-Driven Insights**: Visualize your health metrics including step counts and sleep quality trends.
- **Micro-Sessions**: Quick access to mindfulness and breathwork exercises.

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4
- **Animations**: Motion (framer-motion)
- **Icons**: Lucide React
- **Data Visualization**: Recharts
- **Icons & Components**: Custom minimalist UI components following the Serene Vitality design system.

## Design System

The application follows the **Serene Vitality** design system, documented in `Design.md`. This system emphasizes:
- **Tone**: Calm, encouraging, professional.
- **Colors**: Soft Teal (#4DB6AC) and Off-white (#FBF9F8).
- **Typography**: Manrope for high legibility and modern feel.

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
